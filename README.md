# Maquette de base d’images forestières

## Présentation
Ce projet est une **maquette fonctionnelle** de présentation pour une base de données d’images autour de deux univers forestiers. Il s’agit d’un prototype visuel conçu pour démontrer une direction artistique, une organisation d’interface et une logique de superposition d’images.

L’objectif est de proposer une base de travail claire et professionnelle avant toute phase de développement produit, d’intégration métier ou de mise en production.

## Statut du projet
- **Nature du projet :** maquette / prototype
- **Usage :** démonstration visuelle uniquement
- **Périmètre :** interface front-end et rendu graphique
- **État :** non destiné à une exploitation opérationnelle

## Important
Les images présentes dans cette maquette sont fournies à titre d’illustration uniquement.

Elles **ne sont pas utilisables librement** pour un usage commercial, éditorial ou publicitaire sans vérification préalable des droits associés. Avant toute réutilisation, il est impératif de contrôler :
- les licences d’utilisation,
- les droits d’auteur,
- les éventuelles autorisations de diffusion,
- la compatibilité avec l’usage final prévu.

## Fonctionnalités de la maquette
- Superposition visuelle de plusieurs couches d’images
- Sélection d’une paire visuelle via un contrôle interactif
- Présentation d’un espace de connexion fictif pour illustrer l’usage de la base
- Mise en scène sobre et professionnelle adaptée à une démonstration

## Limites
Cette maquette ne constitue pas un produit fini. Elle ne doit pas être considérée comme :
- une base de données opérationnelle,
- une solution de gestion d’assets,
- une interface de production,
- une garantie de disponibilité ou de conformité des contenus visuels.
